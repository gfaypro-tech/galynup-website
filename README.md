# galynup.fr — CIO Advisory

Site web de **Gaëlle FAY**, fondatrice de GALYNUP, cabinet de conseil CIO Advisory.

## À propos

GALYNUP accompagne les dirigeants et les comités exécutifs dans leurs transformations digitales et leur gouvernance IT. Gaëlle FAY intervient en qualité de **CIO de transition**, **DSI fractionnée** ou **conseil stratégique** pour :

- Définir et piloter la stratégie numérique
- Structurer et transformer la direction des systèmes d'information
- Gouvernance IT et alignement métiers-SI
- Conduite du changement et management de transition
- Préparation aux audits et certifications (ISO, RGPD…)

## Structure du projet

```
galynup-website/

│   └── style.css
├── js/                              # Scripts JavaScript
│   └── script.js
├── php/                             # Scripts PHP backend

│   ├── get-recommendations.php      # Récupération des recommandations

├── prestations/                     # Sous-site PrestaShop

```
